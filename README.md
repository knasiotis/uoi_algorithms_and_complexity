1. Open the *x64 Native Tools Command Prompt for VS* xxxx
2. ```chdir "Google OR Tools"``` 
3. Run the following command ```tools\make run SOURCE=..\src\knapsack.cc```
4. Runtime results are saved in src/results.csv
